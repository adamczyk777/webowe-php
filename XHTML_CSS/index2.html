Odpowiedzi umieszczam tutaj, jako, że jest to spójne logicznie miejsce:

  1. Jaka jest różnica między znacznikami i oraz em? Czy zawsze dają identyczne wyniki?
  2. Jaka jest różnica między znacznikami b oraz strong? Czy zawsze dają identyczne wyniki?
  3. Jaka jest różnica między znacznikami code oraz samp? Czy zawsze dają identyczne wyniki?
  4. Jaka jest różnica pomiędzy fragmentem strony sformatowanej tak: <p>…</p><p>…</p> oraz fragmentem o następującej postaci: <p>…<br/><br/>…</p>?
  1. i oznacza stylowanie tekstu, em oznacza to że tekst ma być wyróżniony np. gdy jest czytany
  2. b oznacza tak jak i stylowanie a strong ze tekst ma byc zaznaczony mocny, np nizszy ton kiedy uzytkownik jest slepy i program mu czyta zawartosc
  3. samp przedstawia przykłądowy output komputera, a code oznacza kod programu, samp jest taki poniewaz kiedys komputery miały fonty typu monospaced
  4. może być różnica w odległości między liniami, ale generalnie br uzywamy jesli intencjonalnie chcemy złamać linię, a nowy tag p uzywamy gdy mamy zupelnie inna mysl i się nie łączy
